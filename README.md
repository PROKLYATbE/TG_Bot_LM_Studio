# TG_Bot_LM_Studio

Репозиторий содержит скрипт для запуска телеграм бота с поддержкой контекста диалога.
Для запуска необходимо предварительно установить <a href="https://lmstudio.ai/">LM Studio</a> и 
загрузить с платформы <a href="https://huggingface.co/">Hugging Face</a> языковую модель.

Также нужно предварительно создать Telegram бота и подключить к проекту.
Создать своего Telegram бота можно с помощью TG бота @BotFather.

## Функциональность бота
* команды: 
   - /start (выводится приветствие и список доступных команд)
   - /model (выводит название используемой LLM)
   - /clear (очищает историю диалога для текущего пользователя)
* запросы пользователя пересылаются LLM, запущенной на этом компьютере, и потом ответ пересылается пользователю

## Особенности бота
* Сохранение контекста диалога для каждого пользователя отдельно
* История диалога хранится в оперативной памяти
* При первом сообщении добавляется системный промпт для настройки поведения модели
* Поддержка независимых диалогов для разных пользователей

## Запуск проекта
1. Установите зависимости: `pip install -r requirements.txt`
2. Настройте токен бота в файле `config.py`
3. Запустите LM Studio и активируйте локальный сервер
4. Запустите бота: `python main.py`
